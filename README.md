## 在 Dify 中使用 Stima API
1. 帳戶圖標 > 設定 > 模型供應商 > 發現更多 Dify 市場
2. 選擇 OpenAI-API-compatible 
3. 進行設置
   - `授權名稱`：填 API 廠商名稱
   - `Model display name`：依你需要的名稱填寫
   - `API Key`：貼上 Key
   - `API endpoint URL`：https://api.stima.tech
   - Model context size
